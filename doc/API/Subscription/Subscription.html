  getRecipientsRules  
getRecipientsRules ( self,  recipients_rule_type=None )

Get the recipient rules objects.

if recipient_rule_type is None, then we return all the recipients rule objects defined for a this given subscription object. If not None we return the given recipients rule objects matching the requested type.

The different types of recipients rule objects : - Role Recipients Rulz - Explicit Recipients Rule - Computed Recipents Rule - Workflow Implied Recipients Rule
